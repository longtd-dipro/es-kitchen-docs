# Order List

## Display Order List by Status

- The order list is displayed in tab format, categorized by status.

- Example:
    - If you respond to an order with a delivery date in the **"Waiting for Delivery Date Response"** status, it will automatically:
        - Move to the **"Waiting for Shipment"** tab
        - Disappear from the **"Waiting for Delivery Date Response"** tab

- Each tab displays the number of new orders (unconfirmed orders).

### Displayed Items

The order list includes the following information:

- Order No.
- Order Date
- Product Name
- Order Status
- Desired Delivery Date
- Scheduled Shipment Date
- Shipment Date
- Quantity
- Tracking Number

### Status Update

- The order status is automatically updated based on each action.

- Example:
    - When a scheduled shipment date is provided, the status automatically changes from:
        - **"Waiting for Delivery Date Response"**
        - → **"Delivery Date Response Provided"**

---

## Search Criteria

You can search orders by:

- Shipping date
- Status

### Available Status Filters

- Waiting for delivery date confirmation
- Waiting for shipment
- Shipped
